-> Um cliente deve possuir um e somente um único cadastro;
-> Um cliente deve possuir um CPF/CNPJ;
-> O cadastro do cliente deve possuir um e-mail válido e uma senha de pelo menos 8 dígitos;

-> O cliente pode ou não vincular uma rede social (Facebook e/ou Google);
-> O cliente pode ou não cadastrar um telefone e e-mail para contato;

-> Um cliente deve possuir pelo menos uma caixa;
-> Cada caixa deverá ter um sensor de temperatura e umidade (DHT11);

-> Cada sensor deve medir as temperaturas e umidades durante o dia, 12 medições (a cada 2 horas);
-> Cada sensor deve pertencer a uma e somente uma caixa, e toda caixa deve ter um e somente um sensor;
-> O cliente pode escolher se quer ver as medições a cada 2, 4, 6, 12 ou 24 horas;

-> Um cliente deve ter pelo menos uma espécie de abelhas;
-> Cada espécie deve ter um nome popular e um nome científico;

-> Um cliente pode ou não fazer uma pergunta;
-> Já existem perguntas pré-inseridas, com respostas predefinidas;
-> O cliente pode responder a perguntas;

-> A média das medições será calculada ao exibir os dados na dashboard;
